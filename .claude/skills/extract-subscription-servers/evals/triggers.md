# Evals для /extract-subscription-servers

## ДОЛЖНО триггерить (positive — минимум 3)

- «Достань сервера из моей подписки» (ссылка на подписку приложена/будет дана).
- «У меня платный VPN, но он не отдаёт список серверов — заглушка/нули. Помоги вытащить.»
- «Подписка зашифрована/привязана к устройству, как достать конфиги серверов?»
- «extract servers from subscription» / «get servers from my VPN subscription».
- «Provider hides the servers, only my app sees them — pull the list».
- «Купил Panterra/NurVPN, хочу завести их сервера к себе — сначала их надо извлечь».
- «Подписка отдаёт какой-то JSON вместо ссылок, ничего не извлекается» (Xray-формат).
- «Сколько у меня серверов в подписке и в каких странах? Вытащи список.»

## НЕ ДОЛЖНО триггерить (negative — минимум 3)

- «Установи мне VPN-панель 3X-UI на сервер» — `/setup-vpn-panel` (установка панели, не извлечение).
- «Настрой маршрутизацию: заведи мои сервера в outbound, сделай балансир, добавь клиентов» — `/configure-vpn-routing` (этот скилл — про настройку панели; извлечение он делегирует сюда или справляется сам для обычной подписки).
- «Сгенерируй клиенту QR-код и vless-ссылку из моей панели» — `/generate-client-config` (генерация конфигов для устройств, не извлечение из подписки провайдера).
- «У меня обычная подписка, ссылки прямо видны в приложении, просто заведи их» — обычная незашифрованная подписка, `/configure-vpn-routing` справится сам (этот скилл — для ЗАКРЫТЫХ/HWID-locked, где список спрятан).
- «Подними прокси на самом сервере (3proxy/sing-box server-side)» — `/setup-server-proxy`.
- «Сделай бэкап / проверь безопасность сервера» — `/setup-backups` / `/audit-security`.

## Edge-cases (опционально)

- «Подписка не работает» — `с уточнением → если «не отдаёт сервера / заглушка / привязка к устройству» → активирует; если «оплата не прошла / приложение не коннектится» → НЕ активирует (это вопрос к провайдеру)`.
- «Вытащи мой VPN-конфиг» — `неоднозначно → уточнить: из подписки провайдера (этот скилл) или из своей панели для клиента (/generate-client-config)?`.
- «Подписка отдаёт нули вместо адресов» — `активирует (это HWID-заглушка — основной кейс скилла)`.
- «Хочу выходить из США через свою подписку» — `пограничное → если сначала надо ИЗВЛЕЧЬ закрытую подписку → этот скилл как первый шаг, затем /configure-vpn-routing; если сервера уже на руках → сразу /configure-vpn-routing`.
