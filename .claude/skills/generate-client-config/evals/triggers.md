# Evals для /generate-client-config

> Источник правды для `description` в SKILL.md.

## ДОЛЖНО триггерить (positive)

- «сделай VPN-конфиг для iPhone»
- «сгенерируй vless-ссылку для клиента alice»
- «дай QR-код для подключения телефона»
- «сделай sing-box JSON для iPhone из моей подписки»
- «выпусти конфиг для клиента bob»
- «нужен QR для жены»
- «client config для iPhone»
- «generate qr code for vpn»
- «сделай конфиг для Hiddify»
- «vless link для мамы»
- «сгенерируй sing-box JSON под мою маршрутизацию РФ-direct»
- «hiddify-конфиг для iPhone с правилами geosite-ru»

## НЕ ДОЛЖНО триггерить (negative)

- «настрой VPN-панель» — это `/setup-vpn-panel`
- «добавь нового клиента в панель» — это `/configure-vpn-routing` (создание клиентов на сервере). Этот скилл — только генерация артефактов из существующих клиентов.
- «настрой прокси на сервере» — это `/setup-server-proxy`
- «удали клиента alice» — обратная операция, не покрывается
- «расскажи про sing-box» — это вопрос про знания, ответ персоны со ссылкой на client-apps.md
- «настрой sing-box на ноутбуке» — это установка sing-box на устройстве, у этого скилла нет SSH-доступа к устройству оператора

## Edge-cases

- «hiddify config с гибкой маршрутизацией» — двусмысленно. Если на стороне **клиента** (sing-box JSON с rule_set) → этот скилл. Если на стороне **сервера** (routing rules в 3X-UI) → `/configure-vpn-routing`. Может попасть в любой из двух.
- «у меня не работает Hiddify на телефоне» — это troubleshooting. Может уместно попасть в этот скилл, если оператор подразумевает «перевыпусти конфиг, может старый битый». В этом случае агент уточняет: «попробуем перевыпуск (запустить меня) или сначала разберём симптомы?»
- «дай мне конфиг» (без контекста) — должен уточнить «для какого клиента и под какую платформу?» (iPhone/Android/desktop). Без уточнения не активируется.
- «конфиг для всех клиентов» — скилл умеет, итерируется по списку клиентов из inbound. Должен дать список и спросить подтверждение «для каждого делать QR + sing-box JSON?»
- «прислать QR в Telegram» — это **не делает** этот скилл (отдельная задача). Скилл выдаёт PNG-файл, оператор сам отправляет.
- «сделай конфиг с правилами по приложениям» — скилл умеет (process_name на desktop, package_name на Android), но для iOS — недоступно (см. platform-quirks.md). Скилл проверяет совместимость и предупреждает.
- «у меня sing-box-vt версия 1.10» — старее нижней планки (1.11). Скилл предупреждает о возможных проблемах и предлагает обновить клиент.
- «нужен конфиг для подписки от провайдера X» — это **другой кейс**: подписка идёт **снаружи** (от провайдера) и просто импортируется в Hiddify напрямую. Скилл говорит «у тебя уже есть подписка от провайдера — импортируй её в Hiddify напрямую через subscription URL, мой скилл нужен для конфигов с **твоей панели**».
