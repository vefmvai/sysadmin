# Evals для /setup-vpn-panel

> Источник правды для `description` в SKILL.md. Сначала пишутся triggers,
> потом из них формулируется description. При расхождении — описание
> обновляется под triggers, не наоборот.

## ДОЛЖНО триггерить (positive — минимум 3)

- «настрой VPN-панель на сервере»
- «поставь 3X-UI»
- «нужен свой VPN»
- «хочу обойти блокировки на своём VPS»
- «настрой свой VLESS на сервере»
- «подними панель управления VPN»
- «setup 3x-ui»
- «installer panel for VLESS Reality»
- «семейный VPN — поставь панель»
- «поставь VPN-панель на VPS в Германии» — distinct case foreign-server
- «3X-UI на российском сервере» — distinct case ru-server

## НЕ ДОЛЖНО триггерить (negative — минимум 3)

- «настрой маршрутизацию VPN» — это `/configure-vpn-routing`, не установка панели
- «добавь клиента в панель» — это `/configure-vpn-routing` (управление клиентами на существующей панели)
- «настрой прокси на сервере чтобы бот видел Claude API» — это `/setup-server-proxy`, не установка панели
- «сделай мне vless-ссылку для iPhone» — это `/generate-client-config`
- «расскажи как работает VLESS+Reality» — это вопрос про знания, ответ персоны со ссылкой на `.claude/knowledge/networking/vpn-protocols.md`
- «у меня уже стоит x-ui, обнови до новой версии» — это `/deploy-service` (update), не setup-vpn-panel
- «удали 3X-UI с сервера» — обратная операция, скилл этого не делает
- «настрой VPN на ноутбуке» — это клиент, не сервер
- «поставь Marzban вместо 3X-UI» — другая панель, скилл работает только с MHSanaei/3x-ui (см. ADR-0005)
- «настрой мониторинг панели через Uptime Kuma» — это `/install-monitoring-stack`

## Edge-cases

- «настрой панель» — неоднозначно (Grafana? 3X-UI? n8n?) → должен уточнить у оператора, не активировать наугад
- «поставь VPN» (без контекста) — должен уточнить «на сервере или клиентское приложение?» перед активацией. На сервере → этот скилл. На клиенте → `/generate-client-config` или ответ персоны со ссылкой на client-apps.md
- «нужен VPN для дома» — уточнить «у тебя есть VPS или нужно сначала арендовать?» → если есть VPS → этот скилл, если нет → персона объясняет про аренду VPS, потом `/bootstrap-new-server`, потом этот скилл
- «у меня форк 3X-UI» — скилл явно отказывает с пояснением (см. `3x-ui-panel.md` §8)
- «настрой свой загр.VPS с VLESS+Reality» — этот скилл с параметром `LOCATION=foreign-server`, активировать
